About This Project

This website is a working example of green IoT monitoring applied to an everyday appliance. It tracks and analyzes my refrigerator’s electricity use (kWh, cost, & daily pattern) via a WiFi IoT switch, processes the result with modern web tools, and presents it as an interactive dashboard for energy awareness, savings, and smart computing education.

  • Real device data profiled and analyzed for insight, not a simulation
  • All code & charts run live in your browser, no backend required
  • Cost, analytics, and reporting—optimized for clarity & modern comfort
  • See Report page for workflow, results, and key findings
